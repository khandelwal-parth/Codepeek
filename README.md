# CodePeek 🔍

A fast, AI-powered code inspector. Paste a URL, drop some code, or upload a screenshot — get the source instantly.

**Live:** [codepeek.vercel.app](https://codepeek.vercel.app)

---

## What it does

- 🌐 **URL Inspector** — paste any website URL and get its source code instantly
- 📋 **Code Paste** — drop any code and view it with syntax highlighting
- 📸 **AI Image Analysis** — upload a screenshot and Gemini Vision recreates the code
- 👁 **Live Preview** — split screen to see exactly what the generated code looks like
- ✏️ **AI Editor** — describe changes and AI edits the code for you
- ⬇️ **Download** — save generated code as an HTML file

---

## Tech Stack

- Vanilla HTML / CSS / JS
- Vercel Serverless Functions (API backend)
- Google Gemini API (Vision + Code generation)
