How to use CodePeek

    

    

      

        
1

        

          
Paste a URL or upload a screenshot

          
Type or paste any public website URL into the search bar, or click the 📷 button to upload a screenshot. You can also Ctrl+V a screenshot directly from your clipboard.

        

      

      

        
2

        

          
Hit Analyze

          
Click the Analyze button. CodePeek auto-detects whether you gave it a URL, raw code, or a screenshot and handles it accordingly — no mode switching needed.

        

      

      

        
3

        

          
Inspect, edit, or preview the code

          
The full source code appears with syntax highlighting. Use the AI Editor to make changes in plain English, toggle the Live Preview to see it render, or just copy/download the code.
